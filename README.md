# opsc-bin
## Dockerfile
This Dockerfile is used for building [cooliron/optimuspy](https://hub.docker.com/repository/docker/cooliron/optimuspy/general) base image.
## Compiled opsc binaries for optimuspy project
Archive, compress and split:
```
tar cf opsc.tar opsc lib
xz -9 -T0 -e opsc.tar --memory=6G
split -b 50M opsc.tar.xz "opsc.tar.xz.part"
```
